## Running Program
+ clone repository and change directory to project
+ Make sure pip is installed
+ linux: pip3 install -r requirements.txt or python3 -m pip install -r requirements.txt
+ windows: py -m pip install -r requirements.txt
+ Run main.py
```
python3 main.py
```
